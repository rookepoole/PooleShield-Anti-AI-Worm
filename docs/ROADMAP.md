# PooleShield Roadmap

## v2.x — Privacy-safe scanner and batch triage

- DAT inspection and extraction
- deterministic batch processing
- chat/log scanning
- policy decisions
- review triage
- local redacted evidence
- privacy bundles

## v3.x — Second-opinion antivirus scanner

- file and folder scanning
- archive scanning
- hash inventory
- extension and entropy risk
- optional YARA-style rule support
- dry-run quarantine planning

## v4.x — Lightweight EDR behavior monitor

- process tree monitoring
- startup/persistence monitoring
- script and command-line risk scoring
- registry and scheduled-task watchlists
- local behavior graph

## v5.x — Controlled containment

- approval-based quarantine
- restore workflow
- block/allow indicators
- incident timeline
- tamper-aware audit log

## v6.x — Optional professional real-time layer

- Windows service
- optional minifilter/driver research
- enterprise update channel
- signed rule updates

## Principle

Build user-mode, auditable, privacy-first protection before any real-time or kernel-level component.
