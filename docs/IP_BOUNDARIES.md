# IP Boundaries

This repository contains a public-safe defensive implementation.

Included:

- PooleShield CLI workflow
- adapters for logs, chat exports, DAT-like files
- scanner/policy/review code
- synthetic examples and tests
- privacy-bundle workflow

Excluded:

- private user logs
- decoded DAT text
- unreleased Poole Math derivations
- unpublished Poole Manifold / Defect Calculus manuscripts
- proprietary benchmark archives
- private research notes
- commercial product strategy not explicitly published here

The license applies only to files in this repository. It does not grant rights to unpublished IP.
