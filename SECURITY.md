# Security Policy

## Defensive scope

PooleShield is a defensive scanner and triage tool. Reports, issues, and pull requests should stay within defensive use.

Do not submit:

- malware samples
- exploit chains
- stolen credentials
- private logs
- raw decoded DAT exports
- secrets, API keys, passwords, or tokens
- instructions for unauthorized access or destructive automation

## Reporting vulnerabilities

Open a GitHub issue with a minimal, non-sensitive reproduction. If sensitive details are required, redact them first.

## Supported versions

This repository is a research prototype. Treat all results as advisory and review before blocking, quarantining, deleting, or executing any file.
