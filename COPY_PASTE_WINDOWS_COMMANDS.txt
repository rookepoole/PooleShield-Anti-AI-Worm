PooleShield v5.3 local test commands
=====================================

1) Extract the package, then open PowerShell in the package folder.

2) Run core checks:

python -m pytest -q
python .\tools\repo_safety_check.py --root .
python .\tools\privacy_leak_check.py --root .

3) Verify safe corpus status:

python .\pooleshield_operator.py safe-corpus-status --dataset .\examples\safe_corpus\tiny_feature_dataset.jsonl --output .\safe_corpus_status_response.json

4) Run the safe corpus benchmark and create a privacy-safe bundle:

python .\pooleshield_operator.py safe-corpus-benchmark `
  --dataset .\examples\safe_corpus\tiny_feature_dataset.jsonl `
  --output-dir .\out\safe_corpus_v5_3 `
  --clean-output `
  --bundle-output `
  --privacy-bundle

5) Upload only this file for review:

out\safe_corpus_v5_3\pooleshield_results_bundle.zip

Never commit out folders, result bundles, local config, local baselines, local history DBs, generated build artifacts, or live malware samples.
