PooleShield v4.4 local test commands
=====================================

1) Extract the package, then open PowerShell in the package folder.

2) Run the core checks:

python -m pytest -q
python .\tools\repo_safety_check.py --root .
python .\tools\privacy_leak_check.py --root .
python .\pooleshield_operator.py desktop --status

3) Install UI dependency:

python -m pip install PySide6

4) Launch desktop prototype:

python .\pooleshield_operator.py desktop

5) If you need a reproducible CLI scan bundle instead of using the UI:

python .\pooleshield_operator.py file-av-scan-baseline `
  --config .\pooleshield_config.json `
  --path "$env:USERPROFILE\Desktop\PooleShieldRealScanSmall" `
  --clean-output `
  --bundle-output `
  --privacy-bundle

6) Verify metadata-only result loading:

python .\pooleshield_operator.py results-load `
  --output-dir .\out\file_av_desktop_v4_4 `
  --decision ALLOW_LOG `
  --limit 25 `
  --output .\results_response.json

7) Upload only the generated privacy bundle, not the package ZIP:

out\file_av_desktop_v4_4\pooleshield_results_bundle.zip

Never commit local config, local baselines, local history DBs, out folders, result bundles, or results_response.json.
